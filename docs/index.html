TRMAPI API Reference

JSON API para consultar la Tasa Representativa del Mercado (TRM) en el Servicio Web de la Superintendencia Financiera de Colombia (SFC).

Esta API es una función Lambda que se ejecuta en AWS y a la cual se accede por API Gateway, lo que garantiza su estabilidad, escalabilidad y rapidez.

La función lambda solicita la información al servicio web de la SFC y almacena el valor de la TRM en una tabla de Dynamodb.

Al usar esta API se reduce la carga sobre los servidores de la SFC y se garantiza la disponibilidad del servicio.

Latest Endpoint

GET /latest

Responde con el valor de la TRM que aplica para la fecha actual. Si para la fecha actual no hay valor de la TRM se devuelte el valor más reciente.

validity: boolean false
in query

Indica si se muestra las fechas para las que son válidas un valor.

Time-Series Endpoint

GET /timeseries

Responde con los valores diarios de la TRM que aplican entre dos fechas. El número máximo de días que se puede solicitar es de 365.

validity: boolean false
in query

Indica si se muestra las fechas para las que son válidas un valor.

start_date: string
in query

La fecha inicial del intervalo de tiempo que se desea consultar en formato YYYY-MM-DD.

end_date: string
in query

La fecha final del intervalo de tiempo que se desea consultar en formato YYYY-MM-DD.
